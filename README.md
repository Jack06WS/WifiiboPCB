# WifiiboPCB
This is a PCB which uses the D1 Mini and RFID-RC522 along with the Wifiibo Software to flash NFC amiibo tags.

I am not associated with Xerxes3rd or the Wifiibo Project.

https://github.com/Xerxes3rd/Wifiibo
